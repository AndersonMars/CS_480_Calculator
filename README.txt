How to compile and run the program:

First, verify you have the java JDK installed. If you do not, please go to https://www.oracle.com/java/technologies/downloads/
and download the latest JDK.

Secondly, to compile the program, open the command prompt, and navigate to the folder where the java file is, then enter the following command:
javac Calculator.java

This command will compile the java file and create the class file.

Finally, to run the program, enter the following command:
java Calculator

This command will run the program.
